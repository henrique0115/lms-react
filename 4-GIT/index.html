Git Commands

  1. git --version
  2. git init
  3. rm -rf .git
  4. git config --global user.name "Emanuel Quintino"
  5. git config --global user.email "emanuelquintino@hotmail.com"
  6. git config --global core.editor "code -w"
  7. git config --list --global
  8. git config --list --local
  9. git status (Working Directory, Stage Area, Repository(Log))
  10. git add .
  11. git rm --cached (file name)
  12. git restore .
  13. git restore --staged .
  14. git commit -m "first commit"
  15. git commit --amend -m "new message"
  16. git reset --soft HEAD~1
  17. git reset --hard HEAD~1
  18. git log
  19. git log -n 3
  20. git diff
  21. git branch -M main
  22. git remote add origin (add address)
  23. git remote --v (consult remote address)
  24. git push -u origin main
  25. git push
  26. git pull
  27. git clone (add link)
  28. git branch
  29. git branch (name (dev))
  30. git branch -D (name)
  31. git checkout (name branch)
  32. git merge (name branch)
  33. git push origin (branch name)
  34. git rm -r --cached . (clear cached Git and remove file GitHub)
  • U (Untracked): São os arquivos que ainda não são monitorados pelo GIT.
  • M (Modified): Arquivos que já foram commitados e agora foram modificados.
  • A (Added): Arquivos que ainda não foram commitados mas já rodou git add.
  • D (Deleted): São arquivo que foram removidos.
